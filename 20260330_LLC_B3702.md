# B3702 [语言月赛202301] 华小科的旅行开始了

## 题目描述

[华中科技大学](https://www.hust.edu.cn/)（Huazhong University of Science and Technology, HUST）面积辽阔。其地图可以被划分为 $n$ 行 $m$ 列。用 $(i,j)$ 来表示第 $i$ 行第 $j$ 列的位置。

华小科的旅行开始了，她将从 $(S_x,S_y)$ 出发。在每一个 $(i,j)$ 位置有着任务指引，告诉华小科她下一步应该前往的位置，或者是结束旅行。

请你依次输出华小科的旅行到达的位置信息，数据保证旅行能够在有限步内结束。

## 输入格式

输入的第一行为四个整数 $m,n,S_x,S_y$，分别代表列数、行数和开始的坐标。

接下来 $n$ 行，每行 $2m$ 个整数。第 $i$ 行的第 $2\cdot j-1$ 和第 $2 \cdot j$ 个数为第 $i$ 行第 $j$ 列的“任务指引”，表示到达位置 $(i,j)$ 后下一步将要前往的位置。若两个数均为 $0$，则代表结束旅行。这些数均为 $0 \sim \max \{n, m\}$ 范围内的整数。

## 输出格式

输出若干行，每行两个数 $x,y$，由空格分开，代表华小科的旅行路径经过了 $(x,y)$。

## 输入输出样例 #1

### 输入 #1

```
3 3 1 1
1 2 1 3 2 1
2 2 0 0 2 3
3 2 3 3 1 1
```

### 输出 #1

```
1 1
1 2
1 3
2 1
2 2
```

## 2. 数学小课堂：坐标系与矩阵

在数学里，我们用**坐标**来定位位置。这道题涉及到了一个很基础但非常有用的概念：**矩阵 (Matrix)**。

### 什么是矩阵？
矩阵就是一个矩形的数字阵列。你可以把它想象成一栋大楼的房间号。


* **行 (Row)**：横着的排，从上往下数。
* **列 (Column)**：竖着的列，从左往右数。
* **坐标表示**：我们在数学里用 $(i, j)$ 表示第 $i$ 行、第 $j$ 列。
    * 比如 $(1, 2)$ 代表第 1 行的第 2 个格子。

### 题目里的“双倍数字”
题目说每行有 $2m$ 个整数，这可能会让 8 岁的你有点困惑。其实很简单：
因为每一个格子里的“纸条”都有**两个数字**（去第几行、去第几列），所以程序在读入数据时，每个格子实际上占了两个位置。

## 3. 这道题的解题方法（小向导秘籍）

要写出解决这道题的程序，你可以分三步走：

### 第一步：准备“抽屉”存地图
我们需要一个能够通过“行号”和“列号”立刻找到指令的地方。
* 在编程里，我们可以用两个**二维数组**。
* 比如 `NextRow[i][j]` 存第 $i$ 行第 $j$ 列指向的**行**。
* `NextCol[i][j]` 存第 $i$ 行第 $j$ 列指向的**列**。

### 第二步：把数字放进抽屉
利用循环语句，把题目给的 $n \times m$ 组数字读进去。
> **注意**：读入的时候，每一行有 $2m$ 个数，每两个一组（一个是行，一个是列），记得把它们准确地放进对应的 `NextRow` 和 `NextCol` 抽屉里。

### 第三步：开始旅行（循环执行）
这里我们不知道要走多少步，所以用一个 **`while` 循环** 最方便！

1.  设定当前位置 `nowX = Sx`，`nowY = Sy`。
2.  **开始循环**：
    * **打印**当前的 `nowX` 和 `nowY`。
    * **取出**当前格子的指令：
        * 新的行 `nextX = NextRow[nowX][nowY]`
        * 新的列 `nextY = NextCol[nowX][nowY]`
    * **判断**：如果 `nextX == 0` 且 `nextY == 0`，说明到终点了，**跳出循环**。
    * **更新**：把 `nowX` 变成 `nextX`，把 `nowY` 变成 `nextY`，回到循环开头继续走。

### 3. 给小程序员的 3 个“避坑”锦囊

* **锦囊一：0 还是 1？**
    题目里的坐标是从 $1$ 开始的（比如 $1$ 到 $n$），但在很多编程语言（如 C++ 或 Python）里，表格（数组）通常是从 $0$ 开始数的。
    * *建议*：你可以把表格开大一点点，直接用 $1 \sim n$ 的位置，这样你的脑子就不容易转晕啦！

* **锦囊二：别走太快！**
    在 `while` 循环里，一定要**先打印**当前位置，**再更新**到下一个位置。如果你先更新了，起点 $(S_x, S_y)$ 就会被漏掉没打印出来。

* **锦囊三：内存要够大**
    题目说 $n, m$ 最大是 $1000$，所以你的地图表格至少要能装下 $1000 \times 1000$ 个格子，每个格子还要装两个数字。

### 4. 数学思考：为什么这个循环一定会停？

题目里有一句很重要的话：**“数据保证旅行能够在有限步内结束”**。

在数学上，这代表这个地图是一个**有向无环图（部分）**或者最终指向了一个**死胡同（0, 0）**。
如果地图里有一个“圈”（比如 A 指向 B，B 又指向 A），华小科就会在里面转一辈子。但因为题目保证了会结束，所以我们不用担心程序会运行到电脑冒烟！
