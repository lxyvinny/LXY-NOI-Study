# B2147 求 f(x,n)

## 1. 认识数学符号：算术平方根

首先，我们要知道这个符号 $\sqrt{}$ 是什么意思。

在数学里，它叫**根号**。如果一个数乘以它自己等于另一个数，那个数就是它的平方根。
* 比如：$2 \times 2 = 4$，所以 $\sqrt{4} = 2$。
* 比如：$3 \times 3 = 9$，所以 $\sqrt{9} = 3$。

在这道题里，我们要算的是一连串根号套在一起的结果。

---

## 2. 观察这个“数学套娃”

我们来看看这个复杂的公式：
$$f(x,n)=\sqrt{n+\sqrt{(n-1)+\sqrt{(n-2)+\sqrt{...+2+\sqrt{1+x}}}}}$$



如果你从左往右看，可能会觉得头晕。但如果我们**从最里面**开始看，就像剥洋葱一样，你会发现它是有规律的：

1.  **最底层：** 先算出 $1 + x$，然后给它戴个“根号帽子”：$\sqrt{1+x}$。
2.  **第二层：** 把上面的结果加上 $2$，再整体戴个“根号帽子”：$\sqrt{2 + \text{刚才的结果}}$。
3.  **第三层：** 把上面的结果加上 $3$，再整体戴个“根号帽子”：$\sqrt{3 + \text{刚才的结果}}$。
4.  ... 一直这样加到 $n$ 为止。

---

## 3. 算法小知识：递推（从内向外）

在计算机里，这种一步步往前算的办法叫做**递推**或者**迭代**。

我们可以准备一个“临时盒子”，名字叫 `result`。
* **第一步：** 盒子里的初始值是 $1 + x$。给它开根号，存回盒子。
* **第二步：** 拿数字 $2$ 加上盒子里的值，再开根号，存回盒子。
* **第三步：** 拿数字 $3$ 加上盒子里的值，再开根号，存回盒子。
* ... 
* **最后一步：** 拿数字 $n$ 加上盒子里的值，最后开一次根号。

### 举个简单的例子
如果 $x = 3, n = 2$：
1.  先算最里面的：$\sqrt{1 + 3} = \sqrt{4} = 2$。
2.  再算外面的：$\sqrt{2 + 2} = \sqrt{4} = 2$。
最后结果就是 $2.00$。

## 4. 给你的小贴士

* **循环的顺序：** 因为我们要从最里面的 $1$ 开始加，一直加到 $n$，所以你的循环可以从 $1$ 数到 $n$。
* **保留小数：** 题目要求保留两位小数。在计算机里，这通常需要用到格式化输出（就像给数字穿上一件漂亮的外衣，只露出来两位小数的部分）。
* **数据范围：** 题目里的 $x$ 和 $n$ 都不大（最大是 $10$），所以计算起来会很快，不用担心电脑累着。
